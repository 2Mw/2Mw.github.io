基础——二维数组的传参方式

如果按照一般函数 function(int a[][]) 这样的传参方式是不允许的，只允许function(int a[][3])类似这个样子的传参方式，这样就限制了我们的很多行为，例如处理数组的操作。

基本思想：

通过创建二级指针的方式

int **newMetrix(int m, int n) {
    //此函数是用来创建二维指针数组
    int **a = new int*[m];
    for (int i = 0; i < m; i++)a[i] = new int[n];
    return a;
}

使用new 关键字完成这一个操作。当然我们在给数组初始化为0的时候，应该对一级指针进行操作。

活动安排问题

查看代码

基本思想：

想要安排尽可能多的活动，如果我们先从耗费时长最少的时间开始挑选的话，其中可能会出现浪费时间较多的状况。所以我们应该按照，结束时间的先后顺序，先结束者先安排，以此类推。

注：

贪心算法对于活动安排问题，它总能求得问题的整体最优解，并且使得活动集合的规模最大（可用数学归纳法证明）。
